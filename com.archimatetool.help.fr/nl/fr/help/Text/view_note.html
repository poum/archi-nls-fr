Ajouter une note

Des notes peuvent être ajoutées à la vue pour permettre des anotations visuelles. Une note n'a pas de signification sémantique dans le langage ArchiMate.

Une note peut être ajoutée depuis la palette de la vue:

L'entrée de palette Note


L'exemple suivant montre des éléments annotés avec une Note:

Utiliser une note dans une vue


Un texte de note, une police de caractères et une couleur de remplissage peuvent être définis dans la fenêtre des propriétés.

Double cliquer sur la note ouvre la fenêtre des propriétés, cliquer sur une note sélectionnée vous permet de modifier directement le texte de la note.


Ajouter une connexion de note

Vous pouvez dessiner une ligne de connexion depuis et vers une Note en utilisant l'outil ligne de "connexion de note" dans la palette. Elle n'a aucune signification sémantique contrairement aux autres types de relations ArchiMate. Double cliquer sur la connexion de note ouvre la fenêtre de propriétés.
