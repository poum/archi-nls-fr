Ajouter des images à des objets

Les blocs de canevas, les emplacements d'image et les éléments adhésifs peuvent contenir des images. Les images dans les blocs et dans les éléments adhésifs sont redimensionnés à une largeur et une hauteur maximum de 100 pixels. Les Images peuvent être de n'importe quelle taille mais nous suggérons que vous les laissiez d'une taille raisonablement petite afin de ne pas trop consommmer de ressources.

Pour ajouter une image à l'un de ces objets, ouvrez la fenêtre des propriétés et sélectionnez l'objet. Double-cliquer sur l'objet dans le canevas ouvrira également la fenêtre des propriétés. Trouvez l'onglet image dans la fenêtre des propriétés et sélectionnez le sélecteur d'image via l'option "Définir l'image ...":

Choisir le sélecteur d'image dans la fenêtre des propriétés

Ceci ouvrira la fenêtre de dialogue du sélecteur d'image:

La fenêtre de dialogue du sélecteur d'image

Toutes les images qui sont contenues dans toute modélisation chargée sont affichées dans le sélecteur de telle sorte que vous pouvez les réutiliser. Si vous souhaitez ouvrir un fichier image depuis votre ordinateur, sélectionnez l'option "Ouvrir à partir d'un fichier ...".


Vous pouvez glisser déposer les fichiers images depuis votre bureau vers le canevas.

Supprimer une image d'un objet

Pour supprimer une image d'un objet, choisissez l'option "Supprimer image" depuis le sélecteur des images.
