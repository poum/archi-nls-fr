Créer un nouveau canevas à partir d'un gabarit

Pour créer un nouveau canevas basé sur un gabarit existant, suivez ces étapes:

  1. Choisissez le dossier "Vues" de la modélisation souhaitée dans l'arborescence des modèles. Faites un clic droit dessus puis choisissez "Nouveau->Canevas à partir d'un gabarit ...". Un assistant va s'ouvrir:



  2. Choisissez le gabarit souhaité. Certains sont fournis dans la section "Gabarits installés" tandis que les gabarits créés par l'utilisateur sont listés dans leurs catégories dans la section "Mes gabarits" (voir Enregistrer un canevas en tant que gabarit"). Le nom et la description du gabarit sélectionnés sont affichés dans la fenêtre d'aperçu de la galerie.
  3. Vous pouvez redimensionner les images vignettes dans la galerie en utilisant le contrôle glisseur de redimensionnement.
  4. Si vous souhaitez ouvrir un gabarit dans un fichier qui n'est pas affiché dans l'assistant, choisissez l'élément "Ouvrir ...". Une boîte de dialogue vous permettra de choisir un gabarit "*.archicanvas".
  5. Appuyez sur "Terminer". Un nouveau canevas est créé et ajouté au dossier "Vues" dans l'arvorescence des modèles dans lequel vous pouvez modifier son nom par défaut. La vue elle-même pourra être ouverte afin d'être prête pour accueillir vos modifications.
