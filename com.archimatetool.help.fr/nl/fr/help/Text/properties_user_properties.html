Propriétés utilisateur

Des propriétés utilisateur peuvent être créées et gérées depuis l'onglet "Propriétés" dans la fenêtre des propriétés.


L'onglet Propriétés

Propriétés utilisateur

Cet onglet vous permet d'ajouter des propriétés utilisateurs de votre chois, ou des attributs, à un concept, un modèle, une vue ou un objet. Par exemple, vous pourriez souhaiter ajouter des informations telles que "Coût", "Emplacement", "Durée", "Temps", "Lien", etc. Les propriétés sont stockées sous forme de paire nom/valeur. Des propriétés avec le même nom peuvent apparaître plus d'une fois dans une modèle ou un objet. Les valeurs sont saisies sous forme de texte libre.

Si la valeur d'une propriété commence par "http://", "https://", "file://" ou "ftp://", elle sera affichée en bleue pour indiquer un hyperlien et une icone sera affichée dans la colonne de gauche. Double cliquer sur l'icone ouvrira le lien dans un navigateur. Double-clicking on the icon will open the link in a Browser.

Pour ajouter une nouvelle entrée de propriété utilisateur:

  1. Cliquer sur le bouton "Nouveau" de la barre d'outils à droite du tableau des propriétés ou choisissez "Nouveau" depuis le menu invoqué par clic droit ou encore double cliquez dans une zone vide du tableau
  2. Modifier le nom de la propriété dans la cellule du tableau. S'il existe des propriétés avec ce nom quelque part ailleurs dans la modélisation, vous pourrez en sélectionner une depuis le menu déroulant dans la cellule du tableau



  3. Modifier la valeur de la propriété dans la cellule "Valeur". S'il existe des propriétés existantes ayant cette valeur (correspondant au nom de la propriété) utilisée ailleurs dans la modélisation, vous pouvez en sélectionner une dans la liste déroulante dans l'éditeur de la cellule

Pour modifier le nom d'une propriété utilisateur existante en un nom nouveau ou existant:

  1. Cliquer sur la cellule du nom de la propriété dans le tableau des propriétés et saisissez un nouveau nom
  2. S'il existe des propriétés utilisées ailleurs dans la modélisation, vous pouvez en choisir dans la liste déroulante dans l'éditeur de la cellule

Pour supprimer une entrée de propriété:

  1. Choisir l'entrée de propriété ou les entrées que vous voulez supprimer
  2. Cliquez sur le bouton de la barre d'outils "Supprimer" à droite du tableau des propriétés ou choisissez "Supprimer" dans le menu obtenu par clic droit

Note: Ajouter une nouvelle propriété à un objet dans la fenêtre des propriétés signifie qu'elle devient disponible en tant que propriété réutilisable pour tous les objets dans la même modélisation qui a les propriétés utilisateur. Supprimer une propriété dans la fenêtre des propriétés ne la supprime que de l'objet sélectionné. Si elle est utilisée dans d'autres objets, elle reste disponible.

Réordonner les entrées des propriétés par glisser déposer:

  1. Choisir l'entrée de propriété ou les entrées dans la table des propriétés que vous souhaitez réorganiser
  2. Glisser déposer les entrées dans le tableau des propriétés pour les réorganiser

Trier les noms des propriétés:

  1. Cliquer sur l'en-tête de la colonne "Nom" du tableau
  2. Les noms des propriétés seront triés alphabétiquement

Ajouter une entrée de propriété hyperlien:

  1. Choisir l'entrée de propriété dans le tableau des propriétés que vous souhaitez utiliser comme hyperlien
  2. Modifier la valeur de telle sorte qu'elle contienne une URL qui commence par "http://", "https://" ou "ftp://"
  3. Double cliquer sur l'icone dans la colonne de gauche pour ouvrir le lien dans un navigateur

Ajouter de nouvelles entrées de propriétés en utilisant des noms de propriétés existants:

  1. Cliquer sur le bouton de la barre d'outils "Nouveau multiple ..." à droite du tableau des propriétés, ou choisissez "Nouveau multiple ..." dans le menu invoqué par clic droit
  2. Choisir les propriétés que vous souhaitez ajouter dans la boîte de dialogue



  3. Appuyer sur OK puis modifier les nouvelles valeurs dans le tableau des propriétés

Gérer et consulter globalement les propriétés utilisateur

  1. Cliquer sur le bouton de la barre d'outils "Gérer" à droite du tableau des propriétés ou choisir "Gérer" dans le menu invoqué par un clic droit
  2. La boîte de dialogue "Gestion des propriétés" apparaît en montrant toutes les clefs des propriétés utilisées globalement dans la modélisation et le nombre de fois qu'elles ont été utilisées:



  3. Vous pouvez supprimer des propriétés ici. Toutes les occurences de la propriété et ses valeurs déclarées seront supprimées de tous les objets qui l'utilisent
  4. Vous pouvez renommer les propriétés en modifiant le nom dans la colonne "Nouveau nom". Toutes les occurences du nom de la propriété seront renommés dans tous les objets qui l'utilisent
