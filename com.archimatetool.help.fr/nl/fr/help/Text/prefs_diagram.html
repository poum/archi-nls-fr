Préférences du diagramme


Vue

Taille de la grille
Fixe l'espacement de la grille utilisée dans les vues des diagrammes.

Palettei ouverte à l'ouverture de la vue
si activé, la palette dans une vue sera ouverte quand la vue est ouverte.

Afficher les infobulles dans les vues
Si activé, les infobulles dans les vues seront affichées.

Modifier le nom après avoir créé un nouvel objet depuis la palette
Si activé, quand un nouvel élément est ajouté dans la palette, vous pouvez immédiatement modifier son nom.


Points de vue

Griser les concepts non autorisés dans l'arborescence des modèles
Quand cette option est définie, tous les concepts qui sont désactivés dans un point de vue sont grisés. Pour plus d'inforations, voir Points de vue.

Cacher les concepts non autorisés dans la palette
Quand cette option est définie, tous les concepts qui ne sont pas autorisés dans un point de vue ne sont pas affichés dans la palette. Pour plus d'information, voir Points de vue.

Cacher les concepts non autorisés dans le connecteur magique
Quand cette option est définie, tous les concepts qui ne sont pas autorisés dans un point de vue ne sont pas affichés dans le connecteur magique. Pour plus d'information, voir Points de vue.

Flouter les concepts non autorisés dans une vue
quand cette option est définie, les concepts non autorisés dans un point de vue sont floutés. Pour plus d'information, voir Points de vue.


Comportement du collage spécial

Toujours coller une référence à l'élément copié
Quand cette option est définie, les éléments copiés dans un diagramme référencent toujours les éléments originaux.

Toujours coller une copie de l'élément copié
Quand cette option est définie, les éléments copiés dans un diagramme sont toujours des duplicatas des éléments originaux.

Pour plus d'information, voir Copier, coller, sélectionner et supprimer des éléments.


Comportement du redimensionnement

Les positions des objets fils sont fixées (défaut)
Quand cette option est définie, les objets fils maintiennent leur position absolue quand l'objet parent est redimensionné depuis le coin suypérieur ou gauche. Redimensionner oar le coin supérieur ou gauche va créer un nouvel espace dans l'objet parent.

Les positions des objets fils sont relatifs à celle de leur parent
Quand cette option est définie, les objets fils maintiennent leur position relative par rapport à l'objet parent qiand l'objet parent est redimensionné depuis le coin supérieur ou gauche. Redimensionner par le haut ou le gauche ne créera pas de nouvel espace dans l'objet parent.


Autre

Utiliser l'ancienne méthode pour dessiner les images
Quand cette option est définie, une ancienne méthode pour dessiner les images à l'échelle est utilisée pour les figures des images du canevas. Elle est plus rapide mais la transparence des images n'est pas préservée. Par défaut, ceci doit être désactivé mais cette option est incluse pour des raisons de compatibilité ascendante.

Utiliser un décalage quand la mise en évidence est dessinée sur les moniteurs en haute résolution
Quand cette option est définie, les mise en évidence des figures sont dessinées avec un petit décalage pour compenser sur les moniteurs en plus haute résolution. Ceci n'est normalement nécessaire que sur les appareils Windows avec des affichages en haute résolution.


Apparence


Globales

Ce sont les réglages globaux pour l'application.

Style d'hyphénation des mots
Définit le stype d'hyphénation des mots par défaut pour les figures ArchiMate.

Dur - l'hyphénation ne se produira que pour les sauts de ligne valides.
Doux - l'hyphénation se produira toujours à la fin de l'espace disponible, coupant au milieu d'un mot.
Tronqué - l'hyphénation se produire toujours à la fin de l'espace disponible, tronquant le mot s'il ne rentre pas.

Valeurs par défaut pour les nouvelles figures ArchiMate

Ce sont les valeurs par défaut à utiliser quand on crée une nouvelle ArchiMate figure. Vous pouvez modifier ces réglages individuellement après.

Largeur
Fixe la largeur par défaut pour les nouvelles figyres ArchiMate.

Hauteur
Fixe la hauteur par défaut pour les nouvelles figures ArchiMate.

Alignement du texte
Fixe l'alignement du texte par défaut pour les nouvelles figyres ArchiMate.

Position du texte
Fixe la position du texte par défaut pour les nouvelles figures ArchiMate.

Gradient par défaut
Fixe le gradient par défaut pour les nouveaux éléments de diagramme. Note - les gradients ne sont pas affichés quand un diagramme est exporté dans une image au format SVG.

Vue esquisse

Arrière plan par défaut
Fixe l'arrière plan par défaut pour les nouvelles vues esquisses créées.


Figures par défaut


Des figures alternatives par défaut sont fournies pour dessiner certaines figures dans les vues diagrammes. Cliquer sur une figure pour définir la forme par défaut à utiliser à la création d'une nouvelle figure.
