Gérer les greffons

Vous pouvez installer et désinstaller des greffons Archi avec le gestionnaire des greffons Archi ou manuellement.

Utiliser le gestionnaire des greffons

Pour installer ou désinstaller un greffon Archi, sélectionnez "Gérer les greffons ..." depuis le menu d'aide.

La boîte de dialogue du gestionnaire des greffons Archi va apparaître. À partir de là, vous pouvez voir les greffons installés et leurs détails.

Installer

Pour installer un greffon Archi en utilisant cette méthode, vous devez disposer du greffon dans le format de greffon Archi. C'est un fichier archive (extension .archiplugin ou *.zip) contenant un ou plusieurs fichiers jar et un fichier marqueur. Des exemples de ce type de greffons Archi sont les greffons Collaboration et jArchi Scripting. Si le greffon est un fichier jar, vous devrez l'installer manuellement (voir ci-après).

Pour installer un nouveau greffon, cliquez sur le bouton "Installer nouveau ...". Choisissez le ou les fichiers du greffon que vous voulez installer. Vous pouvez choisir et installer plus d'un greffon si vous le souhaitez. Il vous sera demandé de redémarrer Archi.

Désinstaller

Les greffons installés seront affichés dans la vue en tableau de la boîte de dialogue. Si vous souhaitez désinstaller un ou plusieurs greffons, sélectionnez-les puis cliquez sur le bouton "Désinstaller". Il vous sera demandé de redémarrer Archi.


Installation manuelle

Si le greffon est au format fichier *.jar, vous pouvez simplement copier le ou les fichiers *.jar dans un répertoire spécial "dropins". (Assurez-vous de quitter d'abord Archi avant de faire ceci.)

Le répertoire "dropins" est situé à l'endroit suivant:

Windows: %user.home%/AppData/Roaming/Archi4/dropins

Mac: %user.home%/Library/Application Support/Archi4/dropins

Linux: %user.home%/.archi4/dropins

("%user.home%" indique votre répertoire utilisateur.)

Si le répertoire "dropins" n'existe pas, vous devrez le créer d'abord.
