Ajouter des éléments directement dans l'arborescence des modèles

Pour ajouter de nouveaux éléments ArchiMate directement dans l'arborescence des modèles, sélectionnez l'un des dossiers "Métier", "Application", "Technologie" ou "Connecteurs" et faites un clic droit. Lentrée de menu "Nouveau" vous permet d'ajouter de nouveaux éléments dans l'arborescence:

Ajouter un nouvel élément directement dans l'arborescence des modèles

Quand le nouvel élément est ajouté dans l'arborescence des modèles, il est sélectionné et vous pouvez lui donner un nouveau nom.

Notez qu'il n'est pas possible d'ajouter des relations directement dans l'arborescence des modèles attendues qu'elles ne peuvent être ajoutées qu'en les glissant dans la fenêtre de l'éditeur de la vue (diagramme).
