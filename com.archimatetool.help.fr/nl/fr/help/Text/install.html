Installer et lancer Archi

Télécharger la version requise sur https://www.archimatetool.com

La version Windows dispose d'un installeur. Éxécutez l'installeur pour installer Archi sur votre système. Les versions 7, 8 et 10 64 bits sont prises en charge. L'installeur copie simplement les fichiers du programme dans le répertoire de destination et associe les fichiers *.archimate avec Archi. Un désintalleur est également inclus.

Vous pouvez également installer Archi manuellement avec un zip d'installation. Dézippez ce fichier et exécutez soit le fichier "Archi" ou celui du programme. Sont également inclus certains fichiers batch Windows - "RegisterFileAssociation.bat" et "UnregisterFileAssociation.bat". Le premier de ces fichiers va enregistrer l'extension de fichier *.archimate avec Archi dans la base de registre Windows. Le second fichier batch défera cette association de fichier.

Les versions Mac et Linux sont packagées respectivement dans des fichiers zip et tar.gz. Désarchivez simpmlement le fichier archive que vous avez téléchargé et double-cliquez sur le fichier application "Archi" pour lancer le programme.


Launching Archi

Le nouvel espace de travail vierge d'Archi ressemble à ce qui suit:

L'espace de travail par défaut d'Archi

L'espace de travail est divisé en sous-fenêtres comme suit:

  • La fenêtre de l'arborescence des Modèles. Par défaut, elle est positionnée en haut à gauche avec le titre "Modèles". C'est ici qu'un ou plusieurs modèles ArchiMate peuvent être vus sous forme d'une structure arborescente.
  • La fenêtre PropriétésElle affiche les propriétés de l'objet sélectionné. Les propriétés de l'objet sélectionné peuvent être éditées dans cette fenêtre
  • La fenêtre Vue d'ensemble. Cette fenêtre affiche le contenu d'un diagramme ArchiMate (Vue) en miniature et sert d'outil de navigation dans le diagramme (Vue) seléctionné.
  • La fenêtre Navigateur. Cette fenêtre affiche le concept du modèle sélectionné et toutes ses relations avec les autres concepts du modèle. Elle est utilisée pour naviguer entre les concepts connectés via leurs relations et est utilisée conjointement avec la fenêtre de l'arborescence de modèles.
  • La fenêtre Palette. Quand elle est ouverte, elle affiche une seule fenêtre pour la palette de dessin utilisée dans les vues.
  • La fenêtre Visualiseur. Cette fenêtre affiche le concept du modèle sélectionné et toutes ses relations avec les autres concepts du modèle de façon graphique. C'est l'équivalent graphique du Navigateur.
  • La fenêtre Conseils. Celle-ci affiche de courts textes de conseil relatif à l'objet sélectionné. Par exemple, en sélectionnant un élément de diagramme "Acteur métier" d'ArchiMate affiche un court résumé de la signification de cet objet et de son but. Sélectionner un élément ou survoler un élément dans la palette du diagramme affiche également un conseil dans cette fenêtre.
  • La fenêtre Valideur. Elle fournit des conseils et des astuces pour améliorer votre modèle.

Ces sous-fenêtres peuvent être réarrangées en les tirant vers de nouvelles positions ou en les tirant hors de la fenêtre principale de l'application pour qu'elles en soit détachées.

Les différentes fenêtres peuvent être affichées ou cachées en sélectionnant les éléments de menu appropriés dans le menu "Fenêtre" de la barre de menu principale ou grâce aux boutons de la barre d'outils:

La barre d'outils des fenêtres

Pour réinitialiser l'espace de travail des fenêtres Archi dans son agencement par défaut, sélectionnez "Réinitialiser l'agencement des fenêtres" dans le menu "Fenêtre" du menu principal.

Vous pouvez cacher ou afficher la barre d'outils principale en choisissant "Cacher/Afficher la barre d'outils" du menu "Fenêtre" dans le menu principal.
