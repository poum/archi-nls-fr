Outils de création de la Palette

Mis à part les outils Outils de sélection, il existe d'autres outils disponibles dans la Palette utilisée pour créer de nouveaux concepts ArchiMate, des notes, des groupes et des relations entre les concepts (connexions). Pour ajouter de nouveaux éléments au canevas, choisissez en un et tirez-le dessus ou cliquez sur le canevas.

Relations

Il existe plusieurs types de relations ArchiMate et un élément de Jonction.

Les outils de création des relations dans la palette

Le premier outil est le Connecteur magique, utilisé pour dessiner des connexions. Viennent ensuite les outils de création pour les relations ArchiMate et un élément de Jonction. This is followed by creation tools for ArchiMate relations and a Junction element.

Notes and Groupes

Utilisé pour ajouter une Note, un conteneur de groupe ou une connexion de note à une vue.

Astuce: Appuyez sur la touche Shift en sélectionnant un outil de la Palette pour le conserver sélectionné.
Par défaut, une fois qu'un élément ou une connexion a été tirée sur le canevas, l'outil de sélection par défaut (flèche) est resélectionné dans la palette. Si vous souhaitez conserver l'outil de la palette actuellement sélectionné, maintenez la touche "Shift" pressée quand vous le sélectionnez.
