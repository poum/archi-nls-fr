Recherche et filtrage dans l'arborescence des modèles

Le nombre d'objets dans l'arborescence des modèles peut croître assez considérablement au fur et à mesure que vous travaillez sur votre modélisation. Bien sûr, vous pourriez vouloir ajouter des sous-dossiers dans la structure des dossiers principaux pour vous aider à organiser ces objets. Toutefois, trouver un objet précis dans l'arborescence peut s'avérer rester difficile.

De façon à chercher dans l'arborescence des modèles, une barre de recherche est incluse dans Archi. Elle est accessible en cliquant sur le bouton "Rechercher" dans la barre d'outils de la fenêtre de l'arborescence des modèles. Cliquer sur ce bouton fait apparaître la barre de recherche:

La barre de recherche rendue visible

Au fur et à mesure que vous saisissez du texte dans le champ de la barre de recherche, l'arborescence des modèles se met à jour pour n'afficher que les objets qui correspondent aux critères de recherche saisis dans la barre de recherche. Par défaut, seul le nom des objets est mis en correspondance avec la chaîne de caractères de recherche. Vous pouvez également chercher dans le champ "Documentation" des objets en cochant ceci dans le menu déroulant "Options de filtres" de la barre de recherche:

Checher à la fois dans "Nom" et "Documentation"

Pour effacer la sélection du texte de recherche, cliquez sur l'icone à la droite du texte. Pour effacer les filtres, désélectionnez "Nom" ou "Documentation".


Types d'objets de filtrage

Pour filtrer certains types de concepts ArchiMate, vous pouvez choisir les différents types à inclure dans filtrer/rechercher dans le menu déroulant:

Filtrer certains types d'objets

Pour réinitialiser le filtre de type d'objets, sélectionnez l'élément de menu "Réinitialiser les filtres".


Filtrer les propriétés utilisateur

Pour filtrer les propriétés utilisateur des objets, vous pouvez sélectionner différentes clefs de propriété dans filtrer/rechercher dans le menu déroulant:

Filtrer sur les propriétés utilisateur


Afficher tous les dossiers

Quand vous raffinez votre recherche, l'arborescence des modèles n'affichera que les objets qui correspondent à vos critères de recherche/filtrage (ou rient du tout si aucun objet ne correspond). Ainsi, les dossiers ne comprenant aucun objet fils qui corresponde ne sont pas affichés. Si vous souhaitez cependant afficher ces dossiers vides quand vous cherchez des objets (vous pourriez vouloir glisser déposer des objets provenant d'autres dossiers, par exemple), vous pouvez configurer ce comportement comme option dans le menu de filtrage en sélectionnant "Afficher tous les dossiers".

Pour fermer la barre de recherche et réinitialiser le filtre de recherche, cliquer sur le bouton "Chercher" une nouvelle fois.
