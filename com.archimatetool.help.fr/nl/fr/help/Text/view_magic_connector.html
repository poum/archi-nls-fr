Ajouter de nouvelles relations et de nouveaux éléments à la vue avec le connecteur magique

Connecter un élément à un autre dans une vue dépend de si la relation est autorisée selon la spécification ArchiMate. Par exemple, vous ne pouvez pas connecter une relation d'affectation d'un composant applicatif à un Acteur métier. A moins que vous ne soyez très familier des règles gouvernant les relations dans ArchiMate, il peut être frustrant de trouver les relations autorisées entre un élément et un autre. Le "Connecteur magique" résoud ce problème.

le connecteur magique dans la palette

Le connecteur magique a deux usages - d'abord pour créer une nouvelle connexion autorisée entre un élément et un autre, et ensuite pour créer un nouvel élément et une connexion autorisée entre l'élément source et l'élément nouvellement créé.

Pour créer une connexion entre un élément source et un élément cible en utilisant le connecteur magique suivant ces étapes:

  1. Choisir l'outil Connecteur magique dans la palette
  2. Cliquer sur l'élément source dans la vue
  3. Cliquer sur l'élément cible dans la vue
  4. Un menu surgissant apparaîtra montrant les relations autorisées entre les deux éléments. Choisir le type requis dans le menu. Il est possible de créer une relation de l'élément source vers l'élément cible, mais aussi dans l'ordre inverse de l'élément cible vers l'élément source. Les deux options directionnelles sont séparées dans le menu surgissant.

Dessiner une connexion en utilisant le connecteur magique

Pour créer un nouvel élément et une connexion en une seule opération utilisant le connecteur magique suivant ces étapes:

  1. Choisir l'outil du connecteur magique dans la palette
  2. Cliquer sur l'élément source
  3. Cliquer sur une zone vierge du canevas de la Vue ou sur le symbole d'un Groupe
  4. Un menu surgissant à plusieurs niveaux va apparaître affichant tous les éléments et les relations autorisées entre les deux. Choisir l'élément requis et la connexion depuis le menu

Dans l'exemple suivant, un nouvel élément Rôle métier sera créé en même temps qu'une relation d'affectation connectée à l'élément Acteur métier choisi au départ.
