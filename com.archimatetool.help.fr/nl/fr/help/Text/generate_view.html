Générer une vue depuis un élément

Vous pouvez générer une nouvelle vue de diagramme depuis un ou des éléments ArchiMate sélectionné. Un ou des éléments ArchiMate sélectionnés deviennent alors le point focal de la vue nouvellement générée. Tous les éléments ArchiMate qui sont liés aux éléments sélectionnés sont également à ajoutés à la vue générée en même temps que toutes les connexions.

Cette fonctionnalité vous permet de créer rapidement une nouvelle vue et un point de vue spécificiquement pour un ou plusieurs éléments ArchiMate reliés.

Pour générer une nouvelle vue depuis un ou des éléments sélectionnés:

  1. Assurez-vous que vous avez sélectionné un ou plusieurs éléments ArchiMate dans l'arborescence des modèles ou dans une vue de diagramme.
  2. Sélectionnez l'option "Générez une vue pour ..." depuis le menu contextuel invoqué par clic droit ou depuis le menu principal "Outils".
  3. Dans la fenêtre de dialogue, choisissez le point de vue cible pour la vue générée. La liste des points de vue disponibles est déterminée par le ou les éléments sélectionnés et s'il sont autorisé dans un point de vue. Notez également que les éléments qui sont liés à l'élément ou aux éléments sélectionnés qui ne sont pas autorisés dans le point de vue cible ne seront pas inclus dans la vue générée. Si vous voulez inclure tous les éléments reliés dans la vue cible, ou si vous n'êtes pas sûr, choisissez le point de vue "Aucun". Vous pourrez changer le point de vue plus tard si vous le souhaitez.
  4. Si vous le souhaitez, modifiez le nom de la vue générée.
  5. Facultativement, cochez "Ajoutez toues les connexions disponibles". Si cette option est cochée, Toutes les connexions entre les éléments seront ajoutées à la vue générée. Si elle est décochée, seules les connexions qui sont directement reliées à ou aux éléments sélectionnés sont ajoutées à la vue générée.
  6. Cliquez sur OK

Fenêtre de dialogue de la vue générée

Une nouvelle vue sera créée contenant le ou les éléments sélectionnés et leurs connexions.
