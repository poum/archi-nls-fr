Ajouter un groupe

Les éléments peuvent être regroupés ensemble dans une vue en utilisant un type de conteneur Groupe. Notez que ce n'est pas le même que l'élément Regroupement d'ArchiMate et qu'il n'a pas de signification sémantique autre que d'être un ensemble visuel d'éléments.

Un groupe peut être ajouté depuis la palette de la vue:

L'entrée Groupe de la palette


L'exemple suivant montre des éléments groupés ensemble en utilisant l'élément groupe pour indiquer les différentes couches dans la modélisation:

Exemple de groupes

Un nom de groupe, une police de caractères et une couleur de remplissage peuvent être déterminées dans la fenêtre des propriétés.

Double cliquer sur un groupe ouvre la fenêtre des propriétés, cliquer sur le champ texte du groupe vous permet de modifier directement le texte du Groupe.
