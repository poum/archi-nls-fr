Travailler avec Archi

Quand vous travaillez avec l'application, vous pourriez souhaiter savoir comment les choses fonctionnent généralement.

Fenêtres et onglets

La zone principale d'édition pour les vues est dans la partie située au centre droit de l'application. Les fenêtres et les onglets peuvent être glissés et déposés pour être réarrangés comme vous le souhaitez. Vous pouvez détacher certaines fenêtres de telle sorte qu'elles soient "flottante".

Si jamais vous veniez à vouloir réinitialiser l'agencement des fenêtres et des onglets de l'application dans leur état originel par défaut, sélectionnez "Réinitialiser l'agencement des fenêtres" dans le menu principal "Fenêtre". Vous pouvez également réinitialiser Archi à ses valeurs par défaut via l'élément de menu "Réinitialiser Archi à ses valeurs par défaut ..." du menu "Aide".


Annuler/Refaire (contextuel selon le modèle)

Des commandes Tout annuler et Tout refaire sont disponibles pour chaque action qui est exécutée par l'utilisateur dans Archi.

Les commandes annuler et refaire dépendent du contexte du modèle sélectionné dans l'arbre des modèles ou dans une vue. Cliquer dans une vue ou dans l'arborescence des modèles activera la commande si une action a été réalisée pour ce modèle.


Aides-mémoire

Des aides-mémoire peuvent aider à guider l'utilisateur via une série d'étape de façon à atteindre un but global. Certaines étapes peuvent être exécutées selon l'aide-mémoire, et certaines sont décrites de telle sorte que l'utilisateur puisse finaliser manuellement l'étape. Les aides-mémoire sont disponibles via le menu principal "Aide". Actuellement, Archi fournit deux aides-mémoire - "Créer une vue de cartographie" et "Créer un nouveau modèle".


Obtenir de l'aide

Une aide contextuelle ou complète est disponible via le menu principal "Aide". Dans les fenêtres, appuyer sur la touche "F1" activera l'aide contextuelle.
