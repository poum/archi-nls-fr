Ajouter une référence vers une autre vue

Un symbole de référence de vue agit comme un lien vers une autre vue depuis une vue. C'est un raccourci qui, lorsqu'on double clique sur lui, ouvre la vue liée.

Pour ajouter une référence de vue, tirez un nœud de vue depuis l'arborescence des modèles sur le canevas de la vue cible. Notez que vous ne pouvez pas référencer la même vue depuis elle-même. La police de caractères et la couleur de remplissage du symbole de la référence de la vue peuvent être définis dans la fenêtre des propriétés.
