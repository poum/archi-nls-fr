Rapports Jasper


Jasper Reports est le moteur Java de génération de rapport le plus populaire au monde. Il combine des sources de données et produit des documents parfaits au pixel près qui peuvent être consultés, imprimés ou exportés dans nombre de formats de documents avec son puissant outil de génération de rapport. Archi peut exporter des modélisations dans différents formats en utilisant les gabarits de Jasper Reports.

Cette option est disponible en utilisant l'élément de menu "Rapport->Jasper ..." du menu principal "Fichier". Une fois que la modélisation est sélectionnée dans l'arborescence des modèles ou dans une vue, cet élément de menu est activé. Il faut alors fournir les détails dans la page de l'assistant suivante:

Page 1 de l'assistant Jasper Reports


Puis dans la page suivante de l'assistant, choisissez le gabarit de rapport que vous préférez ainsi que la langue si vous avez fourni des chaînes de caractères de traduction pour le gabarit du rapport:

Page 2 de l'assistant Jasper Reports


Rapport personnalisable

Si vous choisissez le gabarit "Rapport personnalisable" au moment de l'export, vous pourrez alors régler différentes propriétés pour l'agencement du rapport si vous le basez sur le gabarit de modélisation "Rapport personnalisable" au moment de créer une nouvelle modélisation. Voir "Créer une nouvelle modélisation à partir d'un gabarit pour davantage de détails

En configurant certaines propriétés dans la modélisation qui s'appuie sur le gabarit de modélisation "Rapport personnalisable", vous pouvez régler finement le résultat en sortie. Quand vous créez une nouvelle modélisation basée sur le gabarit de modélisation "Rapport personnalisable", il existe une vue appelée "_LISEZ MOI_" qui liste ces propriétés.

Gabarits de rapport utilisateur

Si vous souhaitez vos propres gabarits de rapports, vous pouvez les placer dans le dossier appelé "jasper-reports" dans le dossier de préférence de l'application. Vous pouvez modifier l'emplacement de ce dossier dans les Préférences. Alternativement, vous pouvez manuellement sélectionner un fichier de gabarit ("main.jrxml") depuis l'option combinée "Choisir ..."

Une structure de dossiers d'exemple pour un rapport personnalisé:

C:\Users\Bagpuss\Application Data\Archi\jasper-reports
                                                 |------My Custom Report
                                                                 |------main.jrxml
                                                                 |------style.jrxt
                                                                 |------other files...

Pour davantage d'information sur la création de Rapports Jasper personnalisés, merci de vous référer à la section développement de ce site web "Archi"
