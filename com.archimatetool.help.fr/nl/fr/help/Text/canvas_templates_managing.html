Gérer vos gabarits de canevas

Vous stockez les gabarits de canevas sur votre système de fichiers en tant que fichiers "*.archimate". Ceux-ci peuvent être stockés n'importe dans votre système de fichiers. Archi vous permet de créer des collections utilitaires qui pointe vers ces gabarits. Ceux-ci sont des raccourcis vers les gabarits réels. Pour gérer votre collection de gabarits, suivez ces étapes:

  1. Choisissez le dossier "Vye" de la modélisation choisie dans l'arborescence des modèles puis faites un clic droit sur lui et choisissez "Nouveau->Canevas à partir d'un gabarit ...". Un assistant va s'ouvrir:



  2. Choisir "Manage.." à partir de la section "Gabarits" sur la gauche de l'assistant. Une fenêtre de dialogue va s'ouvir:



  3. Cette boîte de dialogue vous permet d'ajouter, de renommer et de supprimer de nouvelles catégories ainsi qu'ajouter des gabarits à partir des fichiers de votre collection. Vous pouvez également modifier et changer le nom et la description de chacun des gabarits.
  5. Pour ajouter un gabarit à partir d'un fichier, cliquez sur le bouton "Ajouter yn gabarit ...". Choisissez le fichier "*.archicanvas" dans la boîte de dialogue qui apparaît.
  6. Pour ajouter une nouvelle catégorie de gabarit, cliquez sur le bouton "Nouvelle catégorie ...". Fournir un nom pour la catégorie.
  7. Pour ajouter des gabarits dans une catégorie, tirez et déposez une entrée de gabarit depuis le tableau "Gabarits" dans un dossier de catégorie dans l'arborescence des catégories. Notez qu'un gabarit peut apparaître dans plys d'un dossier de catégorie.
