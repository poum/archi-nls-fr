Gestionnaire des spécialisations

Le gestionnaire des spécialisations vous permet de créer, de modifier et de supprimer des spécialisations des concepts ArchiMate.

Contexte

La spécialisation ArchiMate décrit l'utilisation de la spécialisation d'éléments et de relations:

"Une spécialisation est un moyen simple et puissant pour définir de nouveaux éléments ou relations en se basant sur ceux qui existent. Les éléments spécialisés héritent des propriétés des éléments généraux ... Une nouvelle notation graphique peut également être introduite pour un nouveau concept spécialisé, mais de préférence avec une ressemblance avec la notation du concept plus générique; i.e. en ajoutant un icone ou un autre marquant graphique, ou en modifiant un icone existant. Un élément ou une relation spécialisé ressemble fortement à un stéréotype tel qu'utilisé en UML. La notation de stéréotype avec guillements anguleux peut également être utilisée pour signaler un concept spécialisé."

Pour plus d'information, voir les les mécanismes de personnalisation du langage.


Utiliser le gestionnaire des spécialisations

Le gestionnaire des spécialisations est disponible via le menu principal "Outils" et en cliquant sur le bouton "..." à droite de la spécialisation d'un objet dans la fenêtre des propriétés.

Le gestionnaire des spécialisations


Créer une nouvelle spécialisation

Pour créer une nouvelle spécialisation, cliquez sur le bouton "Nouvelle" ou double-cliquez dans le tableau. Fournissez un nom et choisissez le type de concept auquel la spécialisation sera restreinte. Notez que vous ne pouvez pas avoir plus d'une spécialisation avec le même nom et le même type de concept. Si la spécialisation est utilisée dans une modélisation, le nombre d'instances est affiché dans la colonne "Instances".

Supprimer une spécialisation

Pour supprimer une ou plusieurs spécialisations, choisissez les dans le tableau et appuyez sur le bouton "supprimer". Tous les concepts qui référencent la spécialisation supprimée ne les référenceront plus.

Ajouter ou supprimer une image pour une spécialisation

Les spécialisations d'éléments (sauf les Jonctions) peuvent également être associées à une image qui sera affichée quand l'élément spécialisation est utilisé dans une vue. Choisir une image pour l'objet ou effacer l'image à partir du bouton "Image ..." ou en double cliquant sur la boîte de prévisualisation de l'image. Voir ajouter des images à des objets" pour plus de détails.

Appliquer une spécialisation à un concept ArchiMate

Une spécialisation peut être ajoutée à un concept en le sélectionnant dans sa fenêtre des propriétés.
