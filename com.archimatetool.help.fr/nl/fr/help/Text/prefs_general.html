Préférences générales


Fichiers

Ouvrir automatiquement des vues dans une modélisation quand elle est ouverte depuis un fichier
Quand une modélisation est ouverte depuis un fichier, choisir d'ouvrir automatiquement toutes les vues dans la modélisation ou pas.

Automatiquement créer un fichier de sauvegarde (*.bak) lors de l'enregistrement
Si ceci est coché, à chaque fois qu'un fichier .archimate est enregistré, une copie de sauvegarde avec une extension de fichier .bak sera d'abord créée.

Taille de la liste des fichiers récemment ouverts
Définit la taille de la liste des fichiers récemment ouverts mise en cache et affichée via le menu "Fichier->Ouvrir récents". Peut être fixée de 3 à 15.


Arborescence des mmodèles

Afficher en italique les concepts inutilisés
Quand ceci est coché, tous les éléments ou relations qui ne sont pas utilisés dans une vue seront affichés avec une police de caractères en italique.

Mettre à jour les résultats de la recherche au fur et à mesure de la saisie
Quand ceci est coché, les résultats de recherche dans l'arborescence des modèles sont mis à jour au fur et à mesure que vous saisissez (défaut). Si ce n'est pas coché, les résultats de recherche seront mis à jour quand la touche Entrée est utilisée.

Toujours avertir lors de la suppression d'objets
Quand ceci est coché, un avertissement est toujours affiché quand des objets sont supprimés dans l'arborescence des modèles. Si ce n'est pas coché, un avertissement n'est émis quand on supprime des objets que si ces objets sont utilisés dans des vues.


Expressions des étiquettes

Utiliser pour les noms des vues dans le tableau d'analyse
Quand ceci est coché, la partie "Utilisé dans des vues" dans la section "Analyse" utilise une étiquette déterminée par l'expression d'étiquette du dossier parent (si défini).


Autre

Exporter les images à l'échelle double
Quand ceci est coché, les images sont exportées et copiées dans le presse papier à deux fois leur taille normale. Ceci est coché par défaut pour les systèmes d'exploitation Windows quand le zomm est supérieur à 100% et sur les appareils Mac Retina. C'est pour que la qualité des images exportées ne soit pas compromise puisqu'il n'est pas possible de configurer une DPI plus importante dans l'image. Ce réglage n'est habituellement sur les affichages Linux.

Si l'image redimensionnée a besoin d'être à sa taille logique, vous pourriez la modifier dans un éditeur d'image et configurer son DPI au double de sa valeur. Par exemple, sur Mac, en changeant la DPI de 72 à 144.

Utiliser le navigateur interne Edge (Windows seulement)
Quand ceci est coché, le navigateur interne utilisé par Archi pour le système d'aide, l'onglet des conseils et l'aperçu des rapports est Microsoft Edge. Sur Windows 10, la distribution de l'exécutable WebView2 doit d'abord être installée. Il s'agit de l'installeur autonome Evergreen ("Evergreen Standalone Installer") pour x64. Plus d'informations et le le lien de téléchargement sont disponibles ici. Archi aura besoin d'être redémarré si ce réglage est modfié.

Animation

Animer les vues
Inidque si l'animation doit être utilisée dans les vues des diagrammes pour certaines commandes Annuler/Refaire et lorsque le routeur de connexion de la vue est modifiée.

Vitesse (ms)
Définit la vitesse de l'animation pour animer les vues en millisecondes.

Animer le visualiseur
Définit si l'animation est utilisée dans le visualiseur pour l'agencement initial puis lorsque le bouton "Agencement" est cliqué.

Vitesse (ms)
Définit la vitesse de l'animation pour animer le visualiseur en millisecondes.
