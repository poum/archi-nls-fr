Importer une autre modélisation dans la modélisation courante

Il est possible d'importer et de fusionner une modélisation Archi dans la modélisation actuellement sélectionnée. Vous pouvez ensuite maintenir la modélisation importée comme modèlisation de référence que vous pouvez mettre à jour séparemment et réimporter quand c'est nécessaire.

Certaines choses à prendre en considération au moment d'importer une autre modélisation:

  • Les objets de modélisation sont référencés de manière unique par leur type et leur identifiant. Quand on importe une modélisation, ils sont utilisés pour faire correspondre les objets importés avec les objets cibles.
  • Toutes les propriétés utilisateurs pour un objet sont remplacées au moment de leur importation, il n'y a rien à fusionner à ce niveau.
  • Certaines modifications dans la modélisation importée peut affecter les connexions dans les vues si les relations sont modifiées ou supprimées. Elles sont signalées dans la boîte de dialogue d'état après l'importation.
  • Les objets cibles et les dossiers existant ne seront pas supprimés même si ceux-ci sont supprimés dans la modélisation importée.

Importer une autre modélisation Archi

Pour importer une autre modélisation Archi dans la modélisation actuellement sélectionnée, choisir "Importer une modélisation Archi dans la modélisation actuellement sélectionnée" depuis le menu principal "Fichier". La boîte de dialogue apparaîtra:

Importer une autre modélisation

Options d'import

Mettre à jour les objets et dossiers existant
Si cette option est sélectionnée, alors tous les objets et les dossiers cibles existant seront mis à jour lors de l'import.

Mettre à jour les informations et les dossiers de premier niveau de la modélisation
Si cette option est sélectionnée, alors s'il y a des changements dans le nœud de la modélisation (Nom, Documentation, Propriétés), ceux-ci seront mis à jour. S'il y a des changements dans des dossiers de premier niveau (Documentation, Propriétés), ceux-ci seront également mis à jour.

Afficher la boîte de dialogue d'état après l'import
Si cette option est sélectionnée, alors une boîte de dialogue d'état est affichée après l'importation d'une modélisation. Ceci affichera la liste de ce qui a été mis à jour ou modifié.
