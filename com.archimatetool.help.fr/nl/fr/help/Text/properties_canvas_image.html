Images de canevas

Un objet image est un conteneur pour une image. Vous pouvez verrouiller l'objet image si vous voulez le mettre en lecture seule. Les images peuvent être de n'importe quelle taille mais nous vous suggérons de vous limiter à une taille raisonable si vous ne voulez pas consommer trop de ressources.

Quand vous redimensionnez une image à la souris, vous pouvez maintenir la touche Shift appuyée en même temps de façon à conserver ses proportions. Alternativement, vous pouvez sélectionner l'image puis sélectionner l'élément de menu "Vue->Position->Réintialiser les proportions" (également disponible dans la barre d'outils)/


Propriétés

Sélectionner une image dans une vue de canevas signifie que vous pouvez modifier ou consulter les propriétés suivantes dans la fenêtre des propriétés.


L'onglet principal

Propriétés principales pour une image de canevas

Verrouiller: Sélectionner ceci assure que l'image ne peut être déplacée ou modifiée.
Image: Sélectionner une image pour l'objet ou effacer l'image. Voir "Ajouter des images à des objets" pour plus de détails.
Documentation: Un emplacement pour saisir du texte de documentation utilisateur liée à l'Image.

L'onglet Propriétés

Pour davantage d'information concernant la création et la gestion des propriétés utilisateur, voir Propriétés utilisateur.


L'onglet Apparence

Propriétés d'apparence pour une image de canevas

Couleur des bords: Fixe la couleur des bords utilisés pour l'objet sélectionné. Le bouton "Aucune" supprime les bords de l'objet.
Opacité du remplissage: Fixe l'opacité du remplissage de la figure. Varie de 0 à 255.
Opacité de la mise en évidence: Fixe l'opacité de la mise en évidence de la figure. Varie de 0 à 255.
