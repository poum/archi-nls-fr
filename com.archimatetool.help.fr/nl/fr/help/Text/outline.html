La fenêtre Panorama

La fenêtre Panorama agit comme une fenêtre miniature sur la Vue du diagramme sélectionné de façon à faciliter la navigation si le diagramme est trop grand pour tenir dans la fenêtre principale.

Pour ouvrir ou fermer la fenêtre Panorama, choisir l'option dans le menu principal "Fenêtre" ou depuis la barre d'outils principale.

Panorama

Si la Vue qui est en train d'être modifiée est trop grande pour tenir dans la fenêtre de l'application, un panneau de navigation apparaîtra dans la fenêtre Panorama. Tirer ce panneau de navigation fera défiler les objets dans le canevas de dessin.
