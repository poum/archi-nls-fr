Créer une nouvelle modélisation à partir d'un gabarit

Pour créer une nouvelle modélisation en se basant d'un gabarit, suivez les étapes suivantes:

  1. Choisissez l'élément de menu "Fichier->Nouveau->Modélisation à partir d'un gabarit ..." dans le menu principal "FIchier". Un assistant va s'ouvrir:



  2. Choisir le gabarit souhaité. Certains sont fournis dans la section "Gabarits installés" tandis que les gabarits créés par l'utilisateur sont listés dans leurs catégories dans la section "Mes gabarits" (voir Créer un nouveau gabarit"). Le nom et la description du gabarit sélectionné sont affichés dans la fenêtre de prévisualisation de la Galerie.
  3. Vous pouvez avoir un aperçu de toutes les vignettes d'un gabarit en déplaçant votre souris d'un côté à l'autre de l'image d'aperçu dans la Galerie. Vous pouvez également redimensionner les images dans la galerie en utilisant le contrôle glissière de redimensionnement.
  4. Si vous souhaitez ouvrir un gabarit depuis un fichier qui n'est pas affiché dans l'assistant, choisissez l'élément de menu "Ouvrir ...". Une boîte de dialogue fichiers vous permet de choisir un fichier gabarit "*.archimate".
  5. Appuyez sur "Terminer". Une nouvelle modélisation est créée et ajoutée à l'arborescence des modèles. Le nom de la modélisation est préfixé par "(nouveau)".
