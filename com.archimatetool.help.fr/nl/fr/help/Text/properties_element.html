Propriétés des éléments

Sélectionner un élément de la modélisation dans l'arborescence des modèles ou dans une vue signifie que vous pouvez modifier ou consulter les propriétés suivantes dans la fenêtre des propriétés.


L'onglet principal

Modifier les propriétés "Principales" pour un élément ArchiMate

Nom: Le nom de l'élément ArchiMate
Documentation: Un emplacement pour saisir de la documentation utilisateur concernant l'élément ArchiMate

Dans le champ texte "Documentation", les URLs qui commencent par "http://" "https://" ou "ftp://" affichera un hyperlien. Presser la touche Ctrl / Commande modifiera le curseur en une "main" et vous pourrez ouvrir le lien dans un navigateur.


Le connecteur Jonction possède une propriété supplémentaire de type:

La propriété de type de la jonction

Type: Peut être défini en "Et" ou "Ou". Définir ceci modifie également l'icone de l'élément.

L'onglet Propriétés

Pour davantage d'information concernant la création et la gestion des propriétés utilisateur, voir les propriétés utilisateur.


L'onglet Analyse

Affichage de l'"Analyse" d'un élément ArchiMate

Utilisé dans les vues: Un tableau affichant les vues (s'il y en a) dans lesquelles l'élément choisi est utilisé (affiché dans un diagramme). Double cliquer sur une entrée de ce tableau ouvrira la vue et sélectionnera l'élément dans le diagramme.
Relations du modèle: Un tableau affichant les relations (s'il y en a) vers et depuis l'élément choisi dans la modélisation. Double cliquer sur une entrée de ce tableau sélectionnera la relation dans l'arborescence des modèles si elle est ouverte.
