Gérer vos gabarits

Vous pouvez enregistrer des gabarits dans votre système de fichiers en tant que fichiers "*.architemplate". Ils peuvent être stockés n'importe où dans votre système de fichier local. Archi vous permet de créer des collections utilisateur qui pointent vers des gabarits. Ce sont des raccourcis vers les gabarits réels. Pour gerer votre collection de gabarits, suivez ces étapes:

  1. Choisissez l'élément de menu "Fichier->Nouveau->Modélisation depuis un gabarit..." du menu principal "Fichier". Un assistant va s'ouvrir:



  2. Choisissez "Gérer ..." dans la section "Gabarits" sur la gauche de la fenêtre de l'assistant. Une fenêtre de dialogue va s'ouvrir:



  3. Cette fenêtre de dialogue vous permet d'ajouter, de renommer et de supprimer de nouvelles catégories de gabarits ainsi que d'ajouter des gabarits depuis des fichiers à votre collection. Vous pouvez également modifier et changer le nom et la description de chaque gabarit.
  4. Pour ajouter un gabarit depuis un fichier, cliquez sur le bouton "Ajouter un gabarit ...". Choisissez le fichier "*.archimate" dans la fenêtre de dialogue fichier qui apparaît.
  5. Pour ajouter une nouvelle catégorie de gabarits, cliquez sur le bouton "Nouvelle catégorie ...". Founissez un nom pour la catégorie.
  6. Pour ajouter des gabarits à une catégorie, tirez déposez une entrée de gabarit depuis le tableau "gabarits" dans un dossier de catégorie dans l'arborescence des "Catégories". Notez qu'un gabarit peut figurer dans plus d'un dossier de catégorie.
