Propriétés des modèles

Sélectionner le nœud de plus haut niveau dans l'arborescence des modèles signifie que vous pouvez modifier ou consulter les propriétés suivantes dans la fenêtre des propriétés.


L'onglet principal

Modifier les propriétés "Principales" d'une modélisation ArchiMate

Nom: Le nom de la modélisation ArchiMate
Fichier: Le nom de fichier de la modélisation (lecture seule)
Objectif: Un espace pour saisir une description de l'objectif de la modélisation

Dans le contrôle de texte "Objectif", les URLs qui commencent par "http://" "https://" ou "ftp://" s'affichera comme un hyperlien. Appuyer sur la touche Ctrl / Commande transformera le curseur en une "main" et vous pourrez ouvrir le lien dans un navigateur. Pressing the Ctrl / Command key will change the cursor to a "hand" cursor and you can open the link in a Browser.
