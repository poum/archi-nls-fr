Créer un nouveau gabarit

Pour créer un nouveau gabarit, suivez les étapes suivantes:

  1. Créez une nouvelle modélisation Archi ou ouvrez une modélisation existante que vous avez créée et modifiez là de façon à correspondre au gabarit que vous souhaitez.
  2. Assurez-vous que la modélisation que vous souhaitez enregistrer en tant que gabarit est ouverte et sélectionnée dans l'aborescence des modèles.
  3. Choisissez l'élément de menu "Fichier->Enregistrer comme gabarit ..." depuis le menu principal "Fichier". Un assistant va apparaître:
  4. Choose the "File->Save As Template..." menu item from the main menu. A wizard will open:



  5. Dans l'assitant, fournissez un nom de fichier comme emplacement du fichier gabarit, un nom pour le gabarit (différent du nom de la modélisation) et une description.
  6. Choisissez si vous voulez inclure des vignettes des vues dans le gabarit de la modélisation. Si vous choisissez d'inclure des vignettes des vues dans le gabarit, choisissez laquelle des images sera la vignette "clef". Elle sera la première vignette affichée dans l'assistant "Nouvelle modélisation depuis un gabarit"
  7. Cliquez sur "Suivant" pour passer à la page suivante de l'assitant:



  8. Choisissez si vous voulez ajouter le gabarit à votre collection. Votre collection de gabarits est une liste triée en catégories qui sera affichée dans l'assistant "nouvelle modélisation depuis un gabarit". S'il n'existe pas de catégorie disponible à choisir, vous pouvez créer une nouvelle catégorie en cliquant sur le bouton "Nouvelle ..." dans l'assistant.
  9. Appuyez sur "Terminer".

Le gabarit sera enregistré dans votre système de fichier avec une extension "*.archimate". Vous pouvez partager ce gabarit avec d'autres utilisateurs Archi si vous le souhaitez.
