Aperçu de rapport HTML

L'aperçu de rapports HTML vous permet de voir un rapport HTML généré de la modélisation actuellement sélectionnée dans un onglet dans Archi lui-même. La possibilité d'enregistrer le rapport HTML est disponible dans le menu Fichier->. Pour plus d'information, voir Rapports HTML
