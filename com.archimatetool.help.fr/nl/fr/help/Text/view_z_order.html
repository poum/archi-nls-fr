Définir l'ordre de recouvrement des éléments

Les éléments se recouvrant dans une vue peuvent être mis au premier plan ou à l'arrière plan, ou avancés ou reculés. Ceci est également connu en tant que l'ordre "Z". Ceci est accompli en choisissant l'élémént dans une vue et en faisant un clic droit pour invoque les éléments de menu "Ordre". Ces mêmes éléments de menu sont également disponibles depuis le menu principal "Vue".
