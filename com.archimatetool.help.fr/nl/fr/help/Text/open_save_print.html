Ouvrir, enregistrer et imprimer


Ouvrir une modélisation existante

Pour ouvrir une modélisation ArchiMate existante qui a été créée dans Archi, choisir l'élément de menu "Ouvrir ..." depuis le menu principal "Fichier" ou depuis la barre d'outils principale. Si la modélisation est déjà ouverte dans Archi, cette commande n'aura aucun effet.

Vous pouvez également tirer un fichier ".archimate" de votre bureau dans la fenêtre l'arborescence des modèles pour l'ouvrir.

Les versions Windows et Mac d'Archi gère l'ouverture de l'application depuis le bureau en ouvrant les fichiers ayant l'extension de fichier ".archimate".

The Windows and Mac versions of Archi support opening the application from the desktop by opening files with the ".archimate" file extension.


Enregistrer une modélisation

Une modélisation peut être enregistrer sur le disque à l'aide de l'élément de menu "Enregistrer" du menu principal "Fichier" ou depuis la barre d'outils principale. Si la modélisation n'a pas déjà été enregistrée, une invite vous demandera de saisir un nom de fichier.

La commande de menu "Enregistrer sous ..." vous permet de sauvegarder la modélisation avec un nom différent.


Fermer une modélisation dans l'arborescence des modèles

Il est possible d'avoir plus d'une modélisation ArchiMate ouverte en même temps dans l'arborescence des modèles. Vous pourriez souhaiter en fermer une ou modéliser de telle sorte qu'elle n'apparaisse pas dans l'arborescence des modèles. Ceci est accompli en faisant un clic droit sur la modélisation dans l'arborescence des modèles et en sélectionnant l'élément de menu "Fermer la modélisation" ou en choisissant l'élément de menu "Fermer la modélisation" dans le menu principal "Fichier". Fermer une modélisation ne supprime la modélisation et elle peut être réouverte dans Archi à tout moment à l'aide de la commande "Ouvrir". Si la modélisation a été modifiée, un dialogue vous sera présenté pour vous proposer de l'enregistrer avant de la fermer.


Imprimer une vue (diagramme)

Une vue (diagramme) peut être imprimée sur une imprimante connectée. Ceci est disponible depuis la commande de menu "Imprimer ..." dans le menu principal "Fichier". Cette commande de menu n'est seulement disponible si une vue est ouverte.
