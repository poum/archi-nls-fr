Préférences d'apparence, de couleurs et de polices de caractères


Apparence

Activer les thèmes
Si sélectionnée, les thèmes sont activés ou désactivés. Un redémarrage sera nécessaire pour que ce changement soit pris en compte.

Thème
Choisir le thème à utiliser pour Archi. Un redémarrage sera nécessaire pour ce changement prenne pleinement effet.

Utiliser des angles arrondis
Choisir si les angles seront droits ou arrondis.

Afficher la barre d'état
Afficher ou cacher la barre d'état. Un redémarrage sera nécessaire pour que ce changement prenne effet.


Couleurs

Choisir les couleurs par défaut à utiliser pour les élements, les connexions et les dossiers. Ce sont les couleurs qui seront affichées quand un remplissage ou une ligne de couleur sera fixée "par défaut" dans la fenêtre des propriétés pour un élément de diagramme ou une connexion. Les couleurs par défaut peuvent être fixées de manière différenciée pour chacune des installations d'Archi de telle sorte qu'un utilisateur puisse avoir un ensemble de couleurs de remplissage par défaut complètement différent de celui d'un autre utilisateur. Les couleurs ne seront pas enregistrées dans la modélisation .archimate à moins que l'option pour "Enregistrer la couleur de remplissage par défaut pour un nouveau concept dans le fichier .archimate" ne soit cochée. Les couleurs des dossiers sont globales à l'application et ne sont pas enregistrées dans la modélisation.

Dériver les couleurs des lignes des éléments des couleurs de remplissage
Si ceci est coché, la couleur de ligne d'un élément du diagramme est dérivée comme ombre plus sombre que sa couleur de remplissage. Vous pouvez ajuster le niveau de contraste avec le contrôle "Facteur de contraste". Si cette option est activée, les couleurs de ligne de l'utilisateur sont ignorées.

Enregister la couleur de remplissage par défaut pour les éléments dans la modélisation
Si ceci est coché, la couleur par défaut d'un élément est enregistrée dans le fichier de la modélisation puis sera "figée" dans le fichier. Ceci assure que si ce fichier est partagé avec un autre utilisateur, ils verront la même couleur.

Modifier
Modifier la ou les couleurs sélectionnées.

Réinitialiser
Réinitialise la ou les couleurs sélectionnées à leurs valeurs par défaut.

Importer le schéma
Importer un schéma de couleur qui a été exporté précédemment.

Exporter le schéma
Exporte un schéma de couleur en vue de le partager avec d'autres utilisateurs.


Polices de caractères

Fixe la police de caractères par défaut à utiliser pour les éléments et les connexions dans les diagrammes (vues) et les polices de caractères à utiliser pour les contrôles de l'interface utilisateur. Cliquer sur le bouton "Défaut" fixe la police de caractères par défaut pour les éléments sélectionnés. Le bouton "Modifier" vous permettra de fixer la police de caractères pour les éléments sélectionnés.
