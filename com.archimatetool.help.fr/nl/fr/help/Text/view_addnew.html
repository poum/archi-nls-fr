Ajouter de nouveaux éléments à la vue depuis la palette

Pour créer et ajouter de nouveaux éléments à la vue, sélectionnez l'élément requis dans la Palettei et soit cliquez ou tirez le nouveau symbole sur la zone du canevas. Une fois que le symbole a été ajouté au canevas, vous pouvez le redimensionner et le repositionner avec les actions habituelles par tirer glisser.

Ajouter des éléments à la vue les ajoute automatiquement à l'arborescence des modèles

Il est important de comprendre qu'ajouter des éléments (et des relations) à une vue depuis la palette ajoute automatiquement ces éléments à l'arborescence des modèles également. Par exemple, tirer un élément "Acteur métier" sur le caneas crée à la fois un symbole sur la vue appelée "Acteur métier" et également un nœud dans l'arborescence des modèles avec le même nom.

De plus, si vous supprimez un élément dans une vue, l'élément correspondant dans l'arborescence des modèles n'est pas. C'est parce que l'élément peut être référencé dans une autre vue de la modélisation. Pour supprimer complètement l'élément, vous devez le supprimer dans l'arborescence des modèles ou utiliser l'élément de menu invoqué via un clic droit "Supprimer de la modélisation".

Vous pouvez modifier le nom de l'élément directement en cliquant sur la zone de texte d'un symbole. Double cliquer sur le symbole ouvre la fenêtre des propriétés dans laquelle vous pouvez modifier les propriétés de l'élément sélectionné.


Modifier les propriétés des éléments de la vue

Les propriétés d'un élément sélectionné dans une vue peuvent être modifiés dans la fenêtre des propriétés. Pour ouvrir la fenêtre des propriétés, vous pouvez soit double cliquer sur le symbole ou sélectionner le symbole et ouvrir la fenêtre depuis la barre d'outils principale ou depuis le menu principal. Chaque élément dans la vue possède différentes propriétés qui peuvent être définies ou consultées dans la fenêtre des propriétés. Pour plus d'informations, voir la section La fenêtre des propriétés.

Double cliquer sur un élément dans une vue ouvre la fenêtre des propriétés, cliquer simplement sur le champ texte d'un élément déjà sélectionné vous permet de modifier directement le texte de l'élément.


Ajouter de nouvelles relations (Connexions) à la vue depuis la palette

Pour ajouter de nouvelles relations (connexions) à la vue sélectionnée, choisissez l'outil de connexion requis dans la Palette et tirer d'un élément de la vue vers un autre de la même vue.

De la même façon d'ajouter des éléments de la palette, ajouter une relation à une vue ajoute également automatiquement à l'aborescence des modèles, dans le dossier "Relations".

Quand on ajoute une nouvelle connection à une vue, si le même type de relation de modèles existe déjà entre les éléments source et cible, une boîte de dialogues va apparaître et vous donner les options pour référencer cette relation de modéle d'une connexion ou pour créer une nouvelle connexion et une relation du modèles:

Une boîte de dialogue fournit l'option de réutiliser une relation du modèle.


Relations, règles et contrôle.
Les élément ArchiMatei peuvent se connecter à d'autres éléments selon un ensemble donné de relations (connexions). Certaines relations sont permises, d'autres ne le sont pas. Si une relation n'est pas autorisée, le curseur affichera un symbole "Non autorisé", un cercle avec une ligne diagnole. Si la relation est autorisée, cela affichera le symbole d'une "prise". Some relationships are allowed, others are not. If a relationship is not allowed the cursor will show as a "Not Allowed" symbol, a circle with a diagonal line. If a relationship is allowed, it will show as a "plug" symbol.

Ajouter des relations circulaires (connexions)

Vous pouvez ajouter une relation circulaire (connexion) si vous le souhaitez. C'est une relation dont les éléments cible et source sont les mêmes. Vous pouvez créer une connection d'un élément vers lui-même en sélectionnant la relation requise dans la palette, en cliquant une fois sur l'élément puis en cliquant de nouveau sur cet élément:
