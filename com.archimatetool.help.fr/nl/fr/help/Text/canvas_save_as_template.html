Enregistrer un canevas comme gabarit

Pour enregistrer un canevas existant en tant que gabarit, suivez ces étapes:

  1. Créez un nouveay canevas ou ouvrez une modélisation existante contenant un canevas.
  2. Choisissez le canevas dans l'arborescence des modèles, faites un clc droit sur lui puis choisissez "Enregistrer le canevas comme gabarit ...". Un assistant va s'ouvrir:



  3. Dans l'assistant, fournissez un nom de fichier pour la création du ficher de gabarit, un nom pour le gabarit (qui est différent du nom de la modélisation) et une description.
  4. Choisissez si vous voulez inclure une vignette du canevas dans le gabarit.
  5. Cliquez sur "Suivant" pour aller à la page suivante de l'assistant:



  6. Choisissez si vous souhaitez ajouter le gabarit à votre collection. Votre collection de gabarit est une liste triée par catégories qui sera affichée dans l'assistant Nouveau canevas à partir d'un gabarit". S'il n'y a pas de catégorie disponible à choisir, vous pouvez créer une nouvelle catégorie en cliquant sur le bouton "Nouvelle ..." dans l'assistant.
  7. Appuyez sur "Terminer".

Le gabarit sera enregistré sur votre système de fichiers avec une extension "*.archicanvas". Vous pourrez partager ce gabarit avec d'autres utilisateurs d'Archi si vous le souhaitez.
