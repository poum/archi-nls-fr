Afficher une vue en mode plein écran

Sur Windows et Linux, une Vue peut être affichée et modifiée en mode plein écran (ceci n'est pas disponible sur la version Mac OS X d'Archi puisque OS X possède sa propre gerstion du plein écran). Ceci peut être utile pour maximiser la vue à des fins de présentation. Pour ce faire, sélectionner une vue et appuyer sur la touche F11 ou choisissez l'élément de menu "Plein écran" du menu principal "Vue". La vue sera maximisée:

Une vue en mode plein écran

La fenêtre flottante de la palette peut être fermée avec la touche Escape ou le bouton de fermeture de la fenêtre. Faire un clic droit sur la vue en mode plein écran invoque un menu contextuel où la palette peut être montrée si elle n'est actuellement pas visible.
