Propriété des dossiers

Sélectionner un dossier créé par l'utilisateur dans l'arborescence des modèles signifie que vous pouvez modifier ou consulter les propriétés suivantes dans la fenêtre des propriétés.


L'onglet principal

Modifier les propriétés "Principales" d'un dossier

Nom: Le nom du dossier - ceci ne peut être modifié que pour les sous-dossiers créés par l'utilisateur.
Documentation: Un emplacement pour saisir de la documentation utilisateur en lien avec le dossier

Dans le champ texte "Documentation", les URLs qui commencent par "http://" "https://" ou "ftp://" s'affichera en tant qu'hyperlien. En appuyant sur la touche Ctrl / Commande, le curseur va prendre la forme d'une main et vous pourrez ouvrir le lien dans un navigateur.

 

L'onglet Propriétés

Pour davantage d'information concernant la création et la gestion des propriétés utilisateur, voir les Propriétés utilisateur.

 

L'onglet Étiquette

Ici, vous pouvez définir la façon dont les étiquettes des objets fils d'un dossier sont affichés dans l'arborescence des modèles. Par défaut, c'est le nom de l'objet qui sera affiché. En utilisant une expression d'étiquette, vous pouvez choisir ce qui est affiché pour l'élément dans l'arborescence des modèles. Par exemple, vous pourriez vouloir afficher le nom de l'élément et son type. Ou bien, vous pourriez souhaiter afficher la valeur d'une ou plusieurs propriétés. En utilisant une expression d'étiquette, vous pouvez choisir ce qui est affiché pour l'élément dans l'arborescence des modèles. Par défaut, c'est le nom de l'objet qui sera affiché. En créant une expression d'étiquette pour un dossier, ceci n'affecte que ses objets fils et pas le dossier lui-même.

Pour une liste exhaustive des types d'expression, merci de vous référer au wiki Archi.
