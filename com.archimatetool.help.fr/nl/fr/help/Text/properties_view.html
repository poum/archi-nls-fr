Propriétés des vues

Sélectionner une vue dans l'arborescence des modèles ou dans une vue signifie que vous pouvez modifier ou consulter les propriétés suivantes dans la fenêtre des propriétés.


L'onglet principal

Modifier les propriétés "Principales" d'une vue

<
Point de vue:Sélectionnez le point de vue pour la vue. Pour plus d'informations, voir Points de vue
Nom: Le nom de la vue
Documentation: Un emplacement pour saisir de la documentation utilisateur liée à la vue

Dans le champ de texte "Documentation", les URLs qui commencent par "http://" "https://" ou "ftp://" seront affichées comme des hyperliens. En appuyant sur la touche Ctrl / Commande, le curseur prendra la forme d'une main et vous pourrez ouvrir le lien dans un navigateur.


L'onglet Propriétés

Pour plus d'informations concernant la création et la gestion des propriétés utilisateur, voir les Propriétés utilisateur.


L'onglet Apparence

Modifier les propriétés d'"Apparence" pour une vue

Routeur de connexions: Indique le type de routeur de connexions pour la vue tout entière. Les options sont:

Manual - Lignes directes
Manhattan - Routes utilisant un connecteur orthogonal.

Pour plus d'informations, voir Configurer le type de routeur de connexions pour une vue
