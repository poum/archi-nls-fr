Export et réglages

Pour exporter une modélisation Archi dans un fichier au format Open Exchange, choisir l'élément de menu "Fichier->Exporter->Modélisation au format Open Exchange ...".

Options

Inclure l'organisation des dossiers
Si cette option est cochée, alors la structure organisationnelle des dossiers des éléments et des relations de la modélisation sera incluse dans le fichier exporté.

Valider après l'export
Si cette option est cochée, alors le fichier XML exporté sera validé selon le fichier Schema XSD, ce qui constituera le contrôle final de l'opération d'export.

Copier le fichier Schema XSD dans l'emplacement cible
Si cette option est cochée, alors les fichiers XSD du format Open Exchange seront copié au même endroit que le fichier de sortie.

Code de langue
Configurer les deux lettres du code de langue pour les chaines de caractères tel que défini par l'ISO 639.
