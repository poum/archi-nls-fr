La fenêtre du visualiseur

La fenêtre du visualiseur affiche le concept de modélisation sélectionné et toutes ses relations avec d'autres concepts de modélisation d'une façon graphique. C'est l'équivalent graphique du navigateur. Choisissez un élément ou une relation dans l'arborescence des modèles, le navigateur ou une vue de diagramme mettra à jour la sélection dans le visualiseur.

La fenêtre du visualiseur

Pour utiliser le visualiseur, choisissez n'importe quel élément ou relation dans l'arborescence des modèles, dans le navigateur ou dans une vue. Le visualiseur va se mettre à jour pour refléter la sélection courante. Le visualiseur met en évidence le concept sélectionné et affiche toutes les relations et tous les éléments connectés. Double cliquer sur l'un des éléments connectés vous permet "d'aller dans" l'élément de telle sorte qu'il est le concept central focal et toutes ses relations et concepts connectés sont affichés. Vous pouvez également "aller dans" un élément depuis la barre d'outils dans la fenêtre du visualiseur, et également choisir "Retour" pour retourner à l'élément précédent. Sélectionner "Accueil" dans la barre d'outil pour retourner au premier concept sélectionné.

Profondeur

Le visualiseur affichera les concepts connectés au concept sélectionné à une profondeur par défaut de un niveau (connexions immédiates). Pour modifier le niveau de profondeur jusqu'où les connexions sont calculées, choisissez le niveau de profondeur dans le menu déroulant de la fenêtre du visualiseur:

Filtre de point de vue

Vous pouvez filtrer les éléments affichés dans le visualiseur depuis le menu "Filtre de point de vue". Ceci n'affichera que les éléments qui sont autorisés pour un point de vue donné. Pour voir tous les éléments, sélectionnez le point de vue "Aucun". Pour plus d'informations, voir la section Points de vue.

Filtre d'éléments

Vous pouvez filtrer le type d'éléments affichés dans le visualiseur à l'aide du menu "Filtre d'éléments". Celui-ci n'affichera que le type d'éléments sélectionné. Pour voir tous les types d'éléments, choisir l'option "Tous".

Filtre de connexions

Vous pouvez filtrer les relations affichées dans le visualiseur à l'aide du menu "Filtre de relations". Celui-ci n'affichera que le type de relations sélectionné. Pour voir tous les types de relation, choisissez l'option "Tous".

Direction

Cet élément de menu vous permet de choisir si les relations entrantes et sortantes et les éléments qui y sont connectés sont affichés. Les options sont "Les deux", "Entrantes", "Sortantes". "Entrantes" affiche toutes les relations entrantes, "Sortantes" affiche les relations sortantes tandis que "Les deux" affiche à la fois les relations entrantes et sortantes. Ceci est similaire à la fonctionnalité dans le Navigateur.

Options

La première fois qu'un concept qui a le focus est affiché dans la fenêtre du visualiseur, il est disposé avec un agencement par défaut. Si vous souhaitez réagencer, vous pouvez appuyer sur le bouton "Agencement" sur la barre d'outils de la fenêtre du visualiseur.

Vous pouvez enregistrer le contenu du visualiseur sous forme d'un fichier image, ou copier l'image dans le presse papier du système à l'aide du menu contextuel.
