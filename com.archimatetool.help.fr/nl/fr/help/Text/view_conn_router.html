Définir le type de routeur de connexions pour une vue

Par défaut, les connexions sont dessinées sous forme de lignes droites allant d'un objet à un autre. Des points d'inflexion peuvent être ajoutés à une connexion comme détaillé ici. Cependant, il est possible de définir le type de routeur global pour les connexions de telle sorte que les connexions sont routées autour des objets ou sont dessinées orthogonalement.

Le type de routeur de connexions peut être configuré soit à partie du menu principal "Vue->Routeur de connexions" ou en faisant un clic droit dans une vue ou depuis l'onglet "Apparence" dans la fenêtre des propriétés quand le canevas de la vue est sélectionné.

En plus des réglages spécifiques à la vue, le réglage global Utiliser des ancres de connexion orthogonale dans l'onglet Préférences des connexions peut être utilisé pour modifier la façon dont les connexions sont dessinées.

Les types de routeur disponibles sont les suivants:

Manuel

Les connexions sont dessinées avec des lignes droites directes. Des points d'inflexion peuvent être ajoutés par l'utilisateur.

Routeur manuel


Manhattan

Les connexions sont routées orthogonalement. Des points d'inflexion ne peuvent pas être ajoutés par l'utilisateur.

Routeur Manhattan
