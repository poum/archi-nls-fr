Rapports HTML


Archi offre une fonctionnalité de génération de rapports au format web en générant des pages HTML contenant une arborescence des modèles, un tableau résumant les objets du modèles et les vues de la modélisation.

Cette option est disponible depuis l'élément de menu "Rapport->HTML ..." de l'élément de menu principal "Fichier". Une fois que la modélisation est sélectionnée dans l'arborescence des modèles ou dans une vue, cet élément de menu est activé. Choisir un dossier dans lequel exporter la modélisation et ses vues.

Une unique page HTML, "index.html" est générée dans le répertoire de sortie choisi, ainsi que les fichiers et images associées présents dans les vues de la modélisation.
