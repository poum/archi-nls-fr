Le format de fichier d'échange de l'Open Group

Le standard de l'Open Group définit un format de fichier standard qui prend en charge les échanges de modélisations ArchiMate entre les outils. La portée de ce standard est limitée aux éléments et aux relations d'une modélisation ArchiMate. Il exclut les fonctionnalités qui sont spécifiques à un distributeur spécifique, bien qu'il adresse ce qui peut être arrangé. Le format de fichier d'échange contient les informations de modélisation de base dont:

  • Les objets de tous les types ArchiMate avec les attributs de base
  • Les détails des relations entre les objets
  • Les informations des diagrammes (Phase 2)

Le format de fichier d'échange n'est pas destiné à être un format de fichier persistant pour la modélisation elle-même, c'est un mécanisme pour transporter des données d'un outil à un autre. Les données contenues dans le format de fichier d'échange sont destinées à âtre traitées par un outil "parlant le ArchiMate", excluant les inférences sémantiques propriétaires. Une fois que les données de l'instance ont été importées dans un outil ArchiMate, cet outil les enregistrera probablement dans son propre format de fichier propriétaire.
