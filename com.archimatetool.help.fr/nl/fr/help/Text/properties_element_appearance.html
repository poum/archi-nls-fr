Propriétés d'apparence d'un élément

Choisir un élément dans une vue signifie que vous pouvez modifier ou consulter des propriétés additionnelles dans la fenêtre des propriétées. Différents réglages visuels peuvent être appliqués à un élément pour chacune de ses occurences distinctes dans les différentes vues. Par exemple, l'élément "Service applicatif" peut être coloré en bleu sur une vue et en gris dans une autre.


L'onglet Apparence

Modifier les propriétés d'"Apparence" pour un élément ArchiMate dans une vue

Cet onglet n'est disponible que lorsqu'un élément est sélectionné dans une vue.

7
Couleur de remplissage: Configure la couleur de remplissage de l'élément sélectionné. Le bouton "Défaut" définit la couleur de remplissage comme réglage par défaut.
Couleur de ligne:Définit la couleur de la ligne utilisée pour dessiner l'élément sélectionné. Le bouton "Défaut" configure la couleur de ligne comme réglage par défaut. Si ceci est désactivé, c'est parce que les couleurs des lignes sont dérivées de la couleur de remplissage de l'élément, comme défini dans les préférences.
Gradient: Définit la direction utilisée pour dessiner le gradient de remplissage de l'élément sélectionné. Les réglages possibles sont "Aucun", "Haut", "Gauche", "Droit" et "Bas". Note - les gradients ne sont pas affichés quand un diagramme est exporté en tant qu'image au format SVG.
Alignement du texte: Aligne le texte dans l'élément sélectionné à gauche, centré ou à droite.
Position du texte: Aligne le texte de l'élément sélectionné en haut, au milieu, ou en bas.
Police de caractères: Définit la police de caractères du texte de l'élément sélectionné. Le bouton "Défaut" définit la police de caractères au réglage par défaut comme définit dans les préférences.
Couleur de la police de caractères: Définit la couleur de police de caractères utilisée pour le texte dans l'élément sélectionné. Le bouton "Défaut" définit la couleur de la police de caractères conformément au réglage par défaut.
Opacité du remplissage: Définit l'opacité du remplissage du symbole. S'étend de 0 à 255.
Opacité de la mise en évidence: Définit l'opacité de la mise en évidence du symbole. S'étend de 0 à 255.

 

L'onglet étiquette

Ajouter une expression d'étiquette à afficher dans la vue

Cet onglet n'est disponible que lorsqu'un élément est sélectionné dans une vue.

Par défaut, un nom d'élément sera affiché dans toute vue dans laquelle il apparaît. Utiliser une expression d'étiquette par instance de vue, vous pouvez choisir ce qui est affiché pour l'élément da,s ma vue. PAr exemple, vous pourriez souhaiter afficher le nom de l'élément et son type. Ou vous pourriez souhaiter afficher la valeur de l'une ou plusieurs propriétés.

Pour une liste complète de types d'expression, merci de vous référer au wiki Archi.

 

L'onglet Symbole

Définit les propriétés de "Symbole" pour un Équipement informatique

Cet onglet n'est disponible que quand un élément est sélectionné dans une vue et seulement pour certains symboles.

Certains éléments peuvent être représentés par des symboles alternatifs. Ce sont:

  • Composant applicatif
  • Événement applicatif
  • Interface applicative
  • Processus applicatif
  • Service applicatif
  • Événement métier
  • Interface métier
  • Processus métier
  • Service métier
  • Contrainte
  • Équipement informatique
  • Regroupement
  • Événement de mise en œuvre
  • Nœud
  • Exigence
  • Événement technologique
  • Interface technologique
  • Processus technologique
  • Service technologique
  • Chaîne de valeur

Le symbole par défaut à utiliser quand on crée de nouveaux éléments peut être défini dans les préférences.
