Naviguer dans une vue

Panoramique

Si vous choisissez le premier outil de la palette, cliquez quelque part dans la vue pour lui donner le focus puis maintenez la barre d'espace appuyée, le curseur va changer pour prendre la forme d'une main et vous pouvez alors naviguer dans la vue avec la souris. Vous pouvez également faire la même chose en maintenant appuyé le bouton du milieu de la souris.

Utiliser le clavier plutôr que la souris dans une vue

Il est possible de déplacer et de redimensionner des objets sélectionnés dans une vue en utilisant le clavier de l'ordinateur plutôt que la souris. Pour déplacer un objet, appuyez sur la touche point (".") une fois pour faire apparaître le curseur DEPLACEMENT. Utilisez ensuite les flèches pour déplacer les objets puis la touche ENTREE pour valider le déplacement. Pour redimensionner l'objet, appuyez sur cette même touche point (".") jusqu'à ce que le curseur REDIMENSIONNEMENT apparaisse sur la poignée de redimensionnement souhaitée. Appuyez sur ENTREE pour valider le redimensionnement.

Défilement automatique dans une vue

Parfois, vous pourriez vous apercevoir qu'un concept est en dehors de la zone de la vue alors que vous souhaitez établir une connexion avec lui depuis un autre concept (normalement vous utiliseriez les barres de défilement). Pour ce faire, cliquez simplement sur le concept source après avoir sélectionné l'outil de connexion puis déplacez la souris au bord de la zone de vue. Après un petit moment, la vue va défiler automatiquement.

Zoomer une vue

Vous pouvez zoomer et dézoomer une vue de nombreuses façons différentes:

  • Depuis le menu principal "Vue"
  • En utilisant les raccourcis via les compinaisons de touches Ctrl +, Ctrl =, Ctrl - et Ctrl 0 (touche "Commande" sur Mac)
  • Enumeration utilisant la boîte de commande Zoom dans la barre d'outils principale.
  • En maintenant la touche Ctrl enfoncée (touche "Commande" sur Mac) et en utilisant la molette de la souris

En utilisant ceci en combinaison avec la Vue panoramique, vous pouvez facilement naviguer dans de grands diagrammes.

Trouver et remplacer

Il est également possible de trouver et de remplacer des concepts dans une vue en utilisant leurs noms.
