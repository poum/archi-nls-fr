Dossiers et organisation

Un modèle dans Archi est organisé sous forme d'une structure de dossiers représentant les couches d'ArchiMate et les relations des concepts. Un nouveau modèle comporte les dossiers de plus haut niveau suivants:

Métier Contient les éléments dans la couche "Métier" ainsi que tous les sous-dossiers créés par l'utilisateur
Application Contient les éléments dans la couche "Application" ainsi que tous les sous-dossiers créés par l'utilisateur
Technologie Contient les éléments dans la couche "Technologie" ainsi que tous les sous-dossiers créés par l'utilisateur
Motivation Contient les éléments de la catégorie "Motivation" ainsi que tous les sous-dossiers créés par l'utilisateur
Mise en œuvre & Migration Contient les éléments de la catégorie "Mise en œuvre & Migration" ainsi que tous les sous-dossiers créés par l'utilisateur
Autre Contient les éléments de type Emplacement, Regroupement et Jonction ainsi que tous les sous-dossiers créés par l'utilisateur
Relations Contient les relations entre les concepts telles qu'elles ont été créées dans les vues (diagrammes) ainsi que tous les sous-dossiers créés par l'utilisateur
Vues Contient les liens vers les Vues (diagrammes)

Les éléments ArchiMate peuvent être créés ou supprimés directement dans l'arborescence des modèles (voir Ajouter des éléments directement dans l'arborescence des modèles) ou sont automatiquement ajoutés dans le dossier du type approprié lorsque les objets sont dessinés sur le canevas d'une Vue (voir Ajouter des nouveaux éléments à la Vue depuis la palette). Tous les concepts dans un dossier sont automatiquement triés par prdre alphabétique.


Sous-dossiers créés par l'utilisateur

Des sous-dossiers utilisateurs peuvent être créés dans les dossiers principaux de premier niveau. Ceci vous permet d'organiser les concepts de la façon que vous voulez. Pour ajouter un nouveau sous-dossier dans l'arborescence des modèles, choisissez un dossier de premier niveau (ou un sous-dossier créé par l'utilisateur) et faites un clic droit. L'élément de menu "Nouveau" vous permet d'ajouter un sous-dossier à l'arborescence.

Notez qu'un sous-dossier peut seulement contenir des concepts du même type que le dossier parent de plus haut niveau. Par exemple, seuls les concepts du type "Métier" peuvent être créés dans le dossier "Métier" ou dans l'un de ses sous-dossiers.

Vous pouvez également glisser déposer les concepts et les sous-dossiers dans la même branche de dossier mais pas dans des dossiers de types différents.

Pour renommer un sous-dossier dans l'arborescence des modèles, choisir "Renommer" dans le menu principal Modifier ou depuis le menu contextuel après avoir fait un clic droit.
