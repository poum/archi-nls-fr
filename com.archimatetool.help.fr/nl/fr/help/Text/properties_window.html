La fenêtre des propriétés

La fenêtre des propriétés vous permet de modifier les propriétés des objets sélectionnés dans l'arborescence des modèles ou des symboles ou connexions sélectionnés dans une vue. Certaines propriétés (par exemple, le nom) sont partagés entre un concept dans l'arborescence des modèles et ses contre parties dans une vue. Certaines propriétés ne sont pertinentes pour un objet que dans une vue (couleur de remplissage ou police de caractères par exemple).

Pour modifier les propriétés pour les objets sélectionnés dans l'arborescence des modèles ou dans une vue, choisissez le ou les objets puis ouvrir la fenêtre des propriétés soit en double cliquant le nœud de l'arbre ou l'objet vue ou depuis le menu principal "Fenêtre" ou la barre d'outils principale.

Si elle est activée, l'option de menu locale dans le menu déroulant "Agencement en une seule colonne" affichera certains contrôles dans l'onglet Apparence en une seule colonne.


Note - certaines propriétés ne sont disponibles que lorsque l'objet est sélectionné dans une vue (par exemple, la couleur de remplissage ou la taille des lignes).
