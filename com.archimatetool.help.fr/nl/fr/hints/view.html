Une Vue est une partie de la Description d'Architecture qui adresse un ensemble de sujets d'intérêt et est conçue pour des Parties prenantes spécifiques.

Une Vue est définie au moyen d'un Point de vue qui prescrit les concepts, les modèles, les techniques d'analyse et les visualisations qui sont fournis par la Vue.

Les mêmes éléments que ceux de l'arborescence du Modèle peuvent être réutilisées plusieurs fois dans la même Vue ou dans des Vues différentes.
