Une Vue est une partie d'une description d'architecture qui adresse un ensemble de préoccupations corrélées et qui est conçue spécifiquement pour des parties prenantes en particulier.

Une vue est spécifiée au moyen d'un Point de vue qui prescrit les concepts, les modèles, les techniques et les visualisations qui sont fournis par la Vue.

Les mêmes éléments de l'arborescence des modèles peuvent être réutilisés plusieurs fois dans une même vue ou dans différentes vues.
