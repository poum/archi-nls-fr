An Interaction applicative représente une unité de comportement applicatif collectif effectué par (une collaboration de) deux ou plus Composants applicatifs.

Une interaction applicative décrit le comportement collectif qui est exécuté par les composants qui prennent par à une Collaboration applicative. Une interaction applicative peut également définir le comportement visible extérieurement nécessaire pour réaliser un Service applicatif.

Une Collaboration application peut être affectée à une interaction applicative. Une interaction applicative peut réaliser un Service applicatif. Les Services applicatifs et les Services technologiques peut exposer comme service une interaction applicative. Une interaction applicative peuvent accéder à des Objets de données.

Le nom d'une interaction applicative doit clairement identifier une série de comportements applicatifs.

Categorie:

Application. Comportemental.

Exemples:

Gérer des transactions, Création de profils clients, Mise à jour des enregistrements clients
