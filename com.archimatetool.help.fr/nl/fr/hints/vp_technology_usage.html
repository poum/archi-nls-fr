Le Point de vue Usage technologique montre comment les applications sont supportées par les technologies logicielles et matérielles: les services technologiques sont délivrés par les équipements informatiques; les logiciels systèmes et les réseaux sont fournis aux applications. Ce point de vue joue un rôle important dans le monde logique des applications. Il est très utile pour déterminer les exigences de performance et de qualité sur l'infrastructure en se basant sur les demandes des différentes applications qui l'utilise

Parties prenantes:

Architectes applicatifs, d'infrastructure, gestionnaires opérationnels

Sujets d'intérêt:

Dépendances, performance, mise à l'échelle
