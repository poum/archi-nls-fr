Le Point de vue Usage applicatif décrit comment les applications sont utilisées pour supporter un ou plusieurs processus métier et comment ils sont utilisés par d'autres applications. Il peut être utilisé pour concevoir une application en identifiant les services nécessaires pour les processus et les autres applications ou dans la conception de processus métier en décrivant les services qui sont disponibles. De plus, puisqu'il identifie les dépendances des processus métier sur les applications, il peut être utile pour les gestionnaires des opérations responsables de ces processus.

Parties prenantes:

Architectes entreprise, de processus ou applicatifs, gestionnaires des opérations

Sujet d'intérêt:

Cohérence et complétude, réduction de complexité

Objectif:

Conception, Décision

Portée:

Couche muliple/aspect multiple
