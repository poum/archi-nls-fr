An Fonction applicative représente un comportement automatisé pouvant être effectué par un Composant applicatif.

Une fonction applicative décrit le comportement interne d'un composant applicatif. Si ce comportement est exposé à l'extérieur, ceci est fait via un ou plusieurs services.

Une fonction applicative peut réaliser un ou plusieurs Services applicatifs. Les services applicatifs d'autres fonctions applicatives et des Services technologiques peuvent exposer comme service la fonction applicative. Une fonction applicative peut accéder à des objets de données. Un composant applicatif peut être affecté à une fonction applicative (ce qui signifie que le Composant applicatif exécute la fonction applicative).

Le nom d'une fonction applicative doit de préférence être un nom commun dérivé d'un verbe; par exemple, "comptabilité" ou "enregistrement".

Categorie:

Application. Comportemental.

Exemples:

Comptabilité, Facturation, Création de politique, Calcul de prime, Gestion financière.
