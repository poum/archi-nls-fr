Une Interface applicative représente un point d'accès au travers duquel un service applicatif est mis à la disposition d'un utilisateur, d'un autre composant applicatif ou d'un noeud.

Une interface applicative définit comme la fonctionnalité d'un composant peut être accédée par d'autres éléments. Une interface applicative expose des Services applicatifs à l'environnement.

Une interface applicative peut faire partie d'un Composant applicatif. Une interface applicative peut être affectée à des Services applicatifs, ce qui signifie que l'interface expose ces services à l'environnement.

Le nom d'une interface applicative doit de préférence être un nom.

Categorie:

Application. Structure active.

Exemples:

Échange de données de transactions, services web
