Un Composant applicatif représente une encapsulation de fonctionnalités applicatives, aligné avec la structure d'implémentation, modulaire et remplaçable.

Un composant applicatif est une unité autonome. En tant que tel, il est déployable indépendemment, réutilisable et remplaçable. Un composant applicatif exécute une ou plusieurs Fonctions d'une application. Il encapsule son contenu: ses fonctionnalités ne sont accessibles qu'au travers d'un ensemble d'Interfaces applicatives. Les composants applicatifs qui coopèrent sont connectés via des Collaborations applicatives.

Un composant applicatif peut être affecté à une ou plusieurs Fonctions applicatives. Un composant applicatif possède une ou plusieurs Interfaces applicatives qui exposent ses fonctionnalités.

Le nom d'un composant applicatif doit préférablement être un nom commun.

Catégorie:

Application. Structure active.

Exemples:

Système CRM, portail web, application financière, accès aux données clients.
