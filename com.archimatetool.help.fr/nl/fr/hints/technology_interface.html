Une Interface technologique représente un point d'accès au travers duquel un Service technologique est mis à disposition par un Noeud.

Une Interface technologique définit comme les Services technologiques d'un Noeud peuvent être accédés par d'autres Noeuds. Une Interface technologique expose un Service technologique à l'environnement.

Une Interface technologique définit un type de contrat auquel un composant réalisant cette interface doit se conformer.

Une Interface technologique peut faire partie d'un Noeud via la composition, ce qui signifie que ces interfaces sont fournies par ce Noeud et peuvent servir d'autres Noeuds. Une Interface technologique peut être affectée à un Service technologique pour exposer ce Service à l'environnement.

Le nom d'une Interface technologique doit de préférence être un nom.

Catégorie:

Technologique. Structure active.

Exemples:

Logiciel client, Interface de gestion.
