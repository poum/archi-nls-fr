Une Interface métier représente un point d'accès au travers duquel un service métier est mis à disposition de l'environnement.

Une Interface métier expose la fonctionnalité d'un Service métier aux autres Rôles métier ou acteurs. Il y est souvent fait référence en tant que canal (téléphone, Internet, bureau local, etc.). Le même Service métier peut être exposé via différentes interfaces.

Une Interface métier peut faire partie d'un Rôle métier ou d'un Acteur via une Relation de composition et une Interface métier peut servir un Rôle métier.

Une Interface métier expose un Service métier fourni par un Rôle métier ou une Collaboration métier à son environnement. Un Service métier peut également être exposé via différentes interfaces.

Le nom d'une Interface métier doit de préférence être un nom.

Catégorie:

Métier. Structure active.

Exemples:

Téléphone, Messagerie électronique, Centre d'appel, Chat web, Service support.
