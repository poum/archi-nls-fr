Une Collaboration applicative représente un agrégat de deux ou de plusieurs composants applicatifs qui travaillent ensemble pour effectuer un comportement collectif.

Une collaboration applicative spécifie quels composants applicatifs ou autres collaborations applicatives coopèrent pour exécuter une tâche. Le comportement collaboratif, incluant, par exemple, le mode de communication entre ces composants est modélisés par une Interaction applicative. Une collaboration applicative modélise typiquement une collaboration logique ou temporaire entre composants applicatifs et ne peut pas exister sous forme d'entité séparée au sein de l'entreprise.

La collaboration applicative est une spécialisation d'un élément de structure active interne applicative et agrège deux ou plusieurs composants applicatifs coopérant ou autres collaborations applicatives.

Le nom d'une collaboration applicative doit préférablement être un nom commun.

Categorie:

Application. Structure active.

Exemples:

Deux composants applicatifs collaborent pour la gestion des transactions: un composant de compatibilité et un composant de facturation. Cette collaboration exécute la tâche "Gérer les transactions".
