Un Logiciel système représente un logiciel qui fournit ou contribue à un environnement de stockage, d'exécution et d'utilisation de logiciels ou de données déployés en son sein.

Le Logiciel système est une spécialisation d'un Nœud qui est utilisée pour modéliser l'environnement logiciel dans lequel des Artéfacts s'exécutent. Habituallement, un Logiciel système est combiné avec un Équipement informatique représentant l'environnement matériel pour former un Nœud général.

Un Équipement informatique ou un Logiciel système peut être affecté à un autre Logiciel système; par exemple, pour modéliser différentes couches de logiciels s'exécutant les uns au-dessus des autres. Les Logiciels systèmes peuvent être affectés aux Artéfacts, pour modéliser que ces Artéfacts sont déployés sur ces logiciels. Les Logiciels système peuvent réaliser d'autres Logiciels système. Un Nœud peut être composé de Logiciels système.

Le nom d'un Logiciel système doit de préférence peut être un nom faisant référence au type d'environnement d'exécution.

Catégorie:

Technologe. Structure active.

Exemples:

SGBD, Logiciel système DB2, Logiciel de file de messages, système d'exploitation, Serveur d'application J2EE
